== Description ==
The plugin is an addon for  WPBakery Page Builder (formerly Visual Composer).

It creates an image gallery like the one used on Facebook. The plugin is a wonderful image gallery if you have more then 7 images. It shows 3 images per row, two rows and the last image has a +X number that idicates how many images are in this gallery.

Options: Open Image in PrettyPhoto gallery or you can add a specific URL

“Image Gallery Like Facebook” is open source software. 

Demo URL: https://www.cristianiosub.com/bmw-i3-si-ego-prima-impresie-despre-sharing-automobile-electrice/

== Installation ==
1. Upload "image-gallery-like-facebook" in the `/wp-content/plugins/ folder.
2. Activate the module from your admin panel /wp-admin/plugins.php
3. Enjoy!
